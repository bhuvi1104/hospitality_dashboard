# hospitality_dashboard


🏨 Hospitality Analytics Dashboard (Power BI)

📌 Project Overview
This project leverages Power BI Desktop to analyze and visualize hospitality industry data. The goal was to provide actionable insights into hotel performance, customer behavior, and revenue trends. By transforming raw datasets into interactive dashboards, the project demonstrates how data-driven decision-making can improve operational efficiency and enhance guest satisfaction.

🔑 Key Features
- Revenue Analysis: Breakdown of total revenue by room type, location, and booking channel.
- Occupancy Trends: Interactive charts showing occupancy rates across time periods.
- Customer Segmentation: Insights into guest demographics, booking preferences, and repeat visits.
- Seasonality Impact: Visualization of peak and off-season performance.
- KPIs Dashboard: Key metrics such as ADR (Average Daily Rate), RevPAR (Revenue per Available Room), and cancellation rates.

🛠️ Tools & Techniques
- Power BI Desktop for data modeling and visualization
- DAX for calculated measures and KPIs
- Data Cleaning using Power Query
- Interactive Dashboards with slicers and drill-through functionality

📊 Business Impact
The dashboard enables hospitality managers to:
- Identify high-performing properties and optimize pricing strategies
- Track occupancy and revenue trends in real time
- Understand customer behavior to tailor marketing campaigns
- Reduce cancellations and improve booking efficiency
